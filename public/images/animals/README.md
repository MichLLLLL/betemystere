# Où mettre les photos d'animaux

Dépose ici une photo par animal, nommée d'après son nom en anglais (celui
utilisé dans `animalParts.js`), en minuscules et avec des tirets à la place
des espaces/apostrophes.

Exemples :
- "red panda"              → red-panda.jpg
- "wilson's bird-of-paradise" → wilsons-bird-of-paradise.jpg
- "t-rex"                  → t-rex.jpg

Formats acceptés : `.jpg` en priorité, `.png` en repli si le `.jpg`
n'existe pas (le client essaie les deux automatiquement). Si aucune des
deux n'existe, un joli petit encart "photo à venir" s'affiche à la place —
le jeu fonctionne très bien sans image, tu peux compléter petit à petit.

Astuce : le nom exact attendu pour chaque trait est visible dans
`animalParts.js` (variables `base`, `head`, `ears`, `eyes`, `nose`, `legs`,
`feet`, `tail`, `coat`, `colour`). Tu n'as pas besoin de couvrir toute la
liste d'un coup — commence par les animaux qui reviennent le plus souvent.
